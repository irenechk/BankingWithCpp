🏦 Mini Banking System in C++

A simple **Mini Banking System** built using **C++** that demonstrates basic banking operations and financial calculations using **Object-Oriented Programming (OOP)** and **File Handling**.

This project was developed as a **C++ Case Study** to simulate how basic banking operations work in a real-world system.

---

## 🚀 Features

The system allows users to perform multiple banking operations:

✔ Create a new bank account  
✔ Deposit money into an account  
✔ Withdraw money from an account  
✔ Search account using Account Number  
✔ Calculate **Simple Interest**  
✔ Save customer records using **File Handling**  
✔ View saved banking records  

---

## 🧠 Concepts Used

This project demonstrates the following **C++ programming concepts**:

- Object-Oriented Programming (Classes & Objects)
- Conditional Statements
- Functions
- File Handling (`fstream`)
- Input / Output Streams
- Financial Calculations

---

## 📊 Simple Interest Formula Used

The program calculates interest using the formula:


Simple Interest = (Principal × Rate × Time) / 100


---

## 🏗 Program Workflow

1. User creates a bank account  
2. User can deposit or withdraw money  
3. User can search their account using account number  
4. System calculates simple interest  
5. Records are stored in a file (`bank_records.txt`)
